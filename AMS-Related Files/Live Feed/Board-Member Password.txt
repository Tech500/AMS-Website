How to change password?

Looking at the code on line 365:

javascript
var BOARD_HASH = btoa('AMS-Board-2026'); // Change this password!
To change the password, simply replace AMS-Board-2026 with your new password. 

For example:

javascript
var BOARD_HASH = btoa('YourNewPassword');
